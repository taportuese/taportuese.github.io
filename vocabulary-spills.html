VOCABULARY SPILLS

This project was created in collaboration with a Computer Science student to create a website that uses a machine-learning model to identify lyrical content. My role was to create artwork for the site's outputs; each portrait from this series was created for the student in a client-style relationship in which I combined my own personal style and the client's vision. The set of 15 unique illustrations have a consistent muted yet upbeat color palette and a paper texture reminiscient of crumpled up notes, all while celebrating the individuality and personalities of the rappers. Several of the illustrations are now making sales on my Redbubble page!
